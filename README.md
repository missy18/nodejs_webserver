# nodejs_webserver

nodeJSを使ったサーバーを作る。学習用。Lv.1

## Description
### 課題  
- アクセスしたときにいろんなタイプのファイルをきちんと表示できる
- DBとの連携
- EXPRESSは使わない

### タスク
- substringとlengthを使ってアクセス先の分岐を追加していろんな/止まりに対応する
- DBと連携してみる
- config.jsにホストやポートをわけて整理する
